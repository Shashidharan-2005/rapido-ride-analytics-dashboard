# 🚖 Rapido Ride Analytics Dashboard

## 📌 Project Overview

The Rapido Ride Analytics Dashboard is an interactive Business Intelligence solution developed using Power BI to analyze ride-booking operations and generate actionable business insights.

This project focuses on monitoring key performance indicators (KPIs), revenue trends, customer behavior, vehicle performance, booking success rates, and payment patterns. The dashboard enables stakeholders to make data-driven decisions by transforming raw ride-booking data into meaningful visual insights.

---

## 🎯 Business Objective

The objective of this project was to:

- Analyze ride-booking performance.
- Monitor revenue generation trends.
- Track booking completion and cancellation rates.
- Evaluate vehicle category performance.
- Understand customer behavior and satisfaction.
- Improve operational visibility through interactive dashboards.

---

## 📊 Dashboard Highlights

### Key Performance Indicators (KPIs)

The dashboard tracks:

- Total Revenue
- Total Bookings
- Completed Bookings
- Lost Bookings
- Average Customer Rating
- Total Ride Distance
- Revenue Contribution by Vehicle Type
- Payment Method Distribution
